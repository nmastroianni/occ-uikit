Align

The Align component is a set of utility classes that allow you to align content in various ways.

<div>

  <p>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod

  </p>
</div>

Component Options

Class Description
uk-align-left Aligns the element to the left.
uk-align-right Aligns the element to the right.
uk-align-center Aligns the element to the center.
uk-text-left Aligns text to the left.
uk-text-right Aligns text to the right.
uk-text-center Aligns text to the center.
uk-text-justify Justifies text.

Accessibility

Proper text alignment can improve readability and user experience. However, be mindful of justified text, as it can create uneven spacing that may be difficult for some users to read.
