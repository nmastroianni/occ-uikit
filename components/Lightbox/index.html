Lightbox

A component to create a lightbox for images and videos.

Usage

<div data-uk-lightbox>
    <a class="uk-button uk-button-default" href="https://getuikit.com/docs/images/photo.jpg" data-alt="Alt text for the image goes here">Open Lightbox</a>
</div>

Example:

Open Lightbox

Component Options

The lightbox component has a number of options available.

Caption

You can add a caption to a lightbox item.

<div data-uk-lightbox>
    <a class="uk-button uk-button-default" href="https://getuikit.com/docs/images/photo.jpg" data-caption="Caption text goes here">Open Lightbox</a>
</div>
Open Lightbox - With Caption

Accessibility

The Lightbox component is keyboard accessible and provides WAI-ARIA attributes for screen readers.

  • Users can navigate through the lightbox items using the arrow keys.
  • The lightbox can be closed by pressing the escape key.
