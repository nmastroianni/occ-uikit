Animation

The Animation component provides a collection of smooth, CSS-based animations for web pages.

Usage

<div class="uk-animation-fade">...</div>

Examples:

Fade

Scale Up

Scale Down

Slide Top

Slide Bottom

Slide Left

Slide Right

Shake

Component Options

Class Description
uk-animation-fade Fades the element in.
uk-animation-scale-up Scales the element up.
uk-animation-scale-down Scales the element down.
uk-animation-slide-top Slides the element in from the top.
uk-animation-slide-bottom Slides the element in from the bottom.
uk-animation-slide-left Slides the element in from the left.
uk-animation-slide-right Slides the element in from the right.
uk-animation-shake Shakes the element.
uk-animation-reverse Reverses the animation.
uk-animation-fast Speeds up the animation.

Accessibility

While animations can make a page more engaging, they can also be distracting or cause issues for users with motion sensitivities. Use animations sparingly and avoid long or repetitive animations. The prefers-reduced-motion media query can be used to disable animations for users who have requested it in their system preferences.
