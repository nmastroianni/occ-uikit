Slideshow

A component for creating a slideshow.

Usage

<div class="uk-position-relative uk-visible-toggle uk-light" data-uk-slideshow>

    <ul class="uk-slideshow-items">
        <li>
            <img src="https://getuikit.com/docs/images/photo.jpg" alt="" data-uk-cover>
        </li>
        <li>
            <img src="https://getuikit.com/docs/images/dark.jpg" alt="" data-uk-cover>
        </li>
        <li>
            <img src="https://getuikit.com/docs/images/light.jpg" alt="" data-uk-cover>
        </li>
    </ul>

    <a class="uk-position-center-left uk-position-small uk-hidden-hover" href="#" data-uk-slidenav-previous data-uk-slideshow-item="previous"></a>
    <a class="uk-position-center-right uk-position-small uk-hidden-hover" href="#" data-uk-slidenav-next data-uk-slideshow-item="next"></a>

</div>

Accessibility

The Slideshow component adheres to the Carousel WAI-ARIA design pattern.

  • The slideshow container has the aria-roledescription attribute set to "carousel".
  • Each slide has the aria-roledescription attribute set to "slide" and an aria-label.
  • The navigation has the tablist role, and each navigation item has the presentation role.
  • The navigation links have the tab role, with aria-selected and aria-controls attributes.
  • The previous/next navigation has the button role, with an aria-label and aria-controls attributes.
  • When the slideshow has focus, autoplay is paused.
  • Keyboard navigation is supported. Use the tab and shift+tab keys to navigate between the slideshow and other elements on the page. Use the arrow keys, home, and end keys to navigate between slides.
