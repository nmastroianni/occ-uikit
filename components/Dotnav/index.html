Dotnav

A component for creating a dot navigation.

Usage

<ul class="uk-dotnav">
    <li class="uk-active"><a href="#">Item 1</a></li>
    <li><a href="#">Item 2</a></li>
    <li><a href="#">Item 3</a></li>
</ul>

Vertical modifier

A dotnav can be displayed vertically.

<ul class="uk-dotnav uk-dotnav-vertical">
    <li class="uk-active"><a href="#">Item 1</a></li>
    <li><a href="#">Item 2</a></li>
    <li><a href="#">Item 3</a></li>
</ul>

Slideshow

A dotnav can be used to navigate a slideshow.

<div class="uk-position-relative uk-visible-toggle uk-light" data-uk-slideshow>

    <ul class="uk-slideshow-items">
        <li>
            <img src="https://getuikit.com/docs/images/photo.jpg" alt="" data-uk-cover>
        </li>
        <li>
            <img src="https://getuikit.com/docs/images/dark.jpg" alt="" data-uk-cover>
        </li>
        <li>
            <img src="https://getuikit.com/docs/images/light.jpg" alt="" data-uk-cover>
        </li>
    </ul>

    <div class="uk-position-bottom-center uk-position-small">
        <ul class="uk-dotnav">
            <li data-uk-slideshow-item="0"><a href="#">Item 1</a></li>
            <li data-uk-slideshow-item="1"><a href="#">Item 2</a></li>
            <li data-uk-slideshow-item="2"><a href="#">Item 3</a></li>
        </ul>
    </div>

</div>
