Breadcrumb

Create breadcrumbs to show users their location within the site.

Usage

<ul class="uk-breadcrumb">
  <li><a href="#">Home</a></li>
  <li><a href="#">Parent Page</a></li>
  <li><span>Current Page</span></li>
</ul>

Example:

  • Home
  • Parent Page
  • Current Page

Component Options

There are no component options for breadcrumbs.

Accessibility

For accessibility, it is recommended to wrap your breadcrumb in a <nav> element with an aria-label="breadcrumb" attribute. The last item in the breadcrumb trail should have an aria-current="page" attribute. 

<nav aria-label="Breadcrumb">
  <ul class="uk-breadcrumb">
    <li><a href="#">Home</a></li>
    <li><a href="#">Parent Page</a></li>
    <li aria-current="page"><span>Current Page</span></li>
  </ul>
</nav>
