Accordion

The Accordion component allows you to create vertically stacked sections of content that can be revealed one at a time.

Usage

<ul data-uk-accordion>
    <li class="uk-open">
      <a class="uk-accordion-title" href>Item 1</a>
      <div class="uk-accordion-content">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
      </div>
    </li>
    <li>
      <a class="uk-accordion-title" href>Item 2</a>
      <div class="uk-accordion-content">
        <p>Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor reprehenderit.</p>
      </div>
    </li>
    <li>
      <a class="uk-accordion-title" href>Item 3</a>
      <div class="uk-accordion-content">
        <p>Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat proident.</p>
      </div>
    </li>
  </ul>

Example:

  • Item 1

    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

  • Item 2

    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor reprehenderit.

  • Item 3

    Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat proident.

Component Options

Option Description
multiple Set to true to allow multiple accordion items to be open at the same time. By default, only one item can be open at a time.
active Set to the number of the item you want to be open by default (e.g., 0 for the first item, 1 for the second, and so on).
collapsible Set to false if you want to prevent all items from being closed. By default, all items can be collapsed.
duration The speed of the open/close animation in milliseconds. For example, 500 would be half a second.
transition The animation effect to use when opening and closing items. For example, ease-in-out.

Accessibility

Accordions can help make your content more accessible to people with disabilities. By default, UIkit accordions are built with accessibility in mind and include features like:

  • Keyboard Navigation: Users can navigate through the accordion using the tab and arrow keys.
  • Screen Reader Support: The accordion is properly announced to screen readers, including whether an item is open or closed.
  • Reduced Clutter: By hiding content until it's needed, accordions reduce the amount of information on the page at one time, which can be helpful for users with cognitive disabilities.
