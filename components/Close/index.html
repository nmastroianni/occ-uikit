Close

A generic close icon, that can be used for any dismissible element.

Usage

<button type="button" data-uk-close></button>

Example:

Component Options

There is one option available for the close icon.

Large

A close icon can be large.

<button type="button" class="uk-close-large" data-uk-close></button>

Accessibility

The Close component is designed with accessibility in mind. By default, the close icon is a <button> element, which is focusable and can be activated using the Enter or Space keys.
